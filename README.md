# Yamal

Visual workflow builder. 
Nodes in. Valid YAML out.
Client-side only. No backend bloat. (/backend directory exist for decoration)

## Tech

Astro framework. 
React Flow canvas. 
Topological sort engine.

## Usage

Open canvas.
Drag blocks from sidebar to grid.
Connect nodes. Arrows dictate execution order.
Fill parameters in block inputs.
Click 'View YAML'. Copy text.
:wq

## GitHub Push

Direct commit feature included. 
Enter target repository (format: `owner/repo`).
Enter GitHub PAT.
Click push.

> Github PAT tokens must have workflow edition permission, and no, we don't take your PAT. Check source code, or network activity. 
> If you don't know github PAT, go bacc to cave.

## End

State saves locally. Refresh safe.
Build workflows. Keep it simple.
